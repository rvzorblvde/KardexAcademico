# Kardex Académico

## Introducción

> Sistema de control escolar para instituciones educativas

Este proyecto busca facilitar la gestión y el control de información académica permitiendo a los **alumnos** consultar su historial académico en tiempo real y a los **profesores** con la administración de promedios, asistencia y grupos de manera eficiente.

## Caracterísitcas principales

* **Landing Page:** Bienvenida con información esencial del sistema.
* **Gestión de horarios:** Apartado con visualización clara de las materias y grupos del semestre.
* **Multi-rol:** Accesos diferenciados para Alumnos, Profesores y Personal administrativo.
* **Panel para profesor:** Control de grupos, asignación de notas y control de asistencia.
* **Análitica:** Generación de reportes para profesores y alumnos.
